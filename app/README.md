# 🏃‍♂️ FitCount

**FitCount** is a premium fitness tracking application built with **React Native** and **Expo**. This project focuses on high-fidelity UI replication from Figma, robust state management, and real-time activity tracking.

---

## 🚀 Key Features

* **Pixel-Perfect UI:** Meticulously rebuilt from Figma designs to ensure a premium look and feel.
* **Activity Tracking:** Monitor daily steps, calories, and workout progress in real-time.
* **State Management:** Powered by **Zustand** for lightweight and efficient global state handling.
* **Data Persistence:** Uses **AsyncStorage** to securely save user profiles and fitness history locally.
* **Performance Optimized:** Specifically tested and optimized for mid-range hardware like the **Infinix Hot 12i**.

---

## 🛠️ Tech Stack

* **Framework:** [React Native](https://reactnative.dev/) (Expo)
* **State Management:** [Zustand](https://github.com/pmndrs/zustand)
* **Storage:** [AsyncStorage](https://react-native-async-storage.github.io/async-storage/)
* **Navigation:** React Navigation (Stack & Tabs)
* **Styling:** StyleSheet (Flexbox Architecture)

## 📦 Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Abdullatee102/fitcount.git](https://github.com/Abdullatee102/fitcount.git)
    cd app
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Environment Variables:**
    Create a `.env` file in the root directory and add your configuration:
    ```env
    EXPO_PUBLIC_FIREBASE_API_KEY=your_api_key
    EXPO_PUBLIC_FIREBASE_PROJECT_ID=your_project_id
    EXPO_PUBLIC_API_URL=your_api_endpoint
    ```

4.  **Launch the app:**
    ```bash
    npx expo start
    ```
